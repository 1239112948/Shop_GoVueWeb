# a shop web with go(beego) and vue(element-ui)

## how to create database and table

the file named fmyshop.sql is used to create database and tables.

```bash
mysql
mysql> source myshop.sql
```

## how to run

run beego

``` bash
cd ./myshop
bee run
```

run vue

```bash
cd ./shop
npm install
npm run dev
```

## how to enter the homepage

http://localhost:8080/
